Cascading Style Sheets (CSS)

  • CSS (Cascading Style Sheets) is used to style and design HTML elements on web pages.
  • It controls the layout, colors, fonts, spacing, and overall appearance of a website.
  • CSS can be applied in three ways: inline, internal, and external stylesheets.
  • It supports responsive design using media queries to adapt pages to different screen sizes.
  • CSS works closely with HTML and JavaScript to create visually appealing and interactive web pages.
